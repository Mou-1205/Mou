## 精简 i18n：移除多语言

### 需求
只保留中文，去掉英文、日文、繁体中文的语言文件和切换逻辑。

### 改动

| 操作 | 文件 |
|------|------|
| 删除 | `src/i18n/languages/en.ts` |
| 删除 | `src/i18n/languages/ja.ts` |
| 删除 | `src/i18n/languages/zh_TW.ts` |
| 简化 | `src/i18n/translation.ts` — 直接导出 zh_CN，去掉多语言映射 |
| 收窄 | `src/types/config.ts` — `lang` 类型改为 `"zh_CN"` |

### 效果
- 构建产物更小（少打包 3 份翻译）
- 代码更简单，不用维护多语言
- `i18n()` 函数签名不变，现有调用无需修改
