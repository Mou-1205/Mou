## 顶栏重构（2026-06-30 晚间）

### 问题修复

#### 侧栏点击失效
- **根因**: Pio/Live2D 看板娘 iframe（`#l2d-iframe`）模型加载完成后 `pointer-events` 从 `none` 变 `auto`，覆盖左侧栏
- **修复**: `src/config/pioConfig.ts` 设置 `enable: false`，禁用 Pio 组件
- 组件源码保留，需要时可恢复

#### 顶栏白框闪烁
- **根因**: `wallpaper-navbar-transparent.css` 默认 `--nav-bg: var(--card-bg)`（白色），页面加载时 CSS 变量未生效前短暂显示白色背景
- **修复**: 默认值改为 `transparent`，`--nav-shadow` 改为 `none`

#### 顶栏吸附失效
- **根因**: `sticky` 定位在某些滚动容器（疑似 OverlayScrollbars `data-overlayscrollbars-initialize` 属性影响）下失效
- **修复**: 改用 `position: fixed !important`，`#top-row` 加 `height: 4.5rem` 占位防止内容跳动

---

### 顶栏结构重构

#### 导航拆分（Codex 完成）
`Navbar.astro` 拆分为两个独立区域：
- **左侧**: `.navbar-title-link` — Mouy 品牌入口，独立靠左放大
- **右侧**: `#navbar-action-bar` — 导航链接、搜索、主题切换等操作按钮

#### 毛玻璃 action bar（重新设计）
`wallpaper-navbar-transparent.css` 桌面端核心策略：
- `#top-row` / `#navbar-wrapper` / `#navbar` 全部 `pointer-events: none`（穿透点击）
- `.navbar-title-link` / `#navbar-action-bar` 设 `pointer-events: auto`（仅真实按钮可点击）
- action bar 毛玻璃：`blur(32px) saturate(1.8) brightness(1.05)` + 渐变背景 + 多层阴影
- 深色模式独立配色
- hover 时阴影加深微交互

#### 元素高度统一
所有 action bar 内部元素统一为 `2.5rem`（40px）：
- 导航链接（`.btn-plain`）: 原 `2rem` → `2.5rem`
- 搜索框容器 + 按钮: 原 `2.75rem` → `2.5rem`
- 主题切换按钮: 原 `2.75rem` → `2.5rem`
- action bar 整体高度: `3.25rem`（52px），内边距 `0.375rem`

#### CSS 变量优先级
`#navbar > div` 的 `background`/`backdrop-filter`/`box-shadow` 加 `!important`，防止被 `main.css` 的 `.wallpaper-transparent` 规则覆盖

## 顶栏优化（2026-07-01）

### 毛玻璃效果重设计
- **风格**: Mica 哑光材质 + 液态玻璃
- **底色**: `rgba(255,255,255, 0.35)`（亮色）/ `rgba(255,255,255, 0.08)`（暗色）
- **模糊**: `blur(40px) saturate(1.4)` — 重模糊让背景色彩渗透
- **边框**: `1px solid rgba(255,255,255, 0.3)` — 极细色散折射边框
- **阴影**: 柔和弥散 `0 8px 32px rgba(0,0,0, 0.08)` + 内顶部微光
- 品牌入口和 action bar 使用相同样式

### 搜索按钮居中修复
- **问题**: 搜索按钮 `absolute right-0 top-0` 贴在顶部，和其他按钮不对齐
- **修复**: Search.svelte wrapper 去掉 `w-11`，按钮 `absolute right-0`（不加 `top-0`），CSS 强制 `top: 2px !important` 实现垂直居中
- 搜索展开时按钮溢出不影响其他按钮位置

### 导航链接选择性隐藏
- **需求**: 鼠标靠近搜索区域时，只隐藏"关于"和"其他"，保留"主页""归档""我的"
- **实现**: DropdownMenu.astro 加 `data-name={link.name}` 属性，Navbar.astro JS 通过 `data-name="About"` / `data-name="Others"` 精确选择并设置 `opacity: 0`

### 元素尺寸调整
- action bar 高度: `2.75rem` → `3.25rem`，内边距 `0.375rem`
- 内部按钮统一: `2rem` → `2.5rem`
- 字号: `0.75rem` → `0.8125rem`
