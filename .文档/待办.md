## 时间线（timeline.ts）

需要填写真实的时间线数据，当前已清空。

### 数据格式

```typescript
{
  id: "唯一标识",              // 随便起，不重复就行
  title: "标题",               // 显示的大标题
  description: "描述",         // 详细说明
  type: "类型",                // 四选一：
                               //   education — 上学/学习
                               //   work — 工作/实习
                               //   project — 项目
                               //   achievement — 成就/获奖
  startDate: "2026-06-30",     // 开始时间
  endDate: "2026-07-02",       // 结束时间（可选，不填就是"至今"）
  location: "地点",            // 可选
  organization: "组织/学校",   // 可选
  skills: ["技能1", "技能2"],  // 可选
  achievements: ["成就1"],     // 可选
  icon: "material-symbols:school", // 图标
  color: "#059669",            // 颜色
  featured: true,              // 是否高亮显示（可选）
}
```

### 简单来说

- 🆔 起个 id
- 📝 写标题和描述
- 🏷️ 选个类型（上学/工作/项目/成就）
- 📅 填时间
- 🎨 选个颜色和图标
